    Membuat webserver di openshift

    • Buat akun di openshift dan isikan username serta password kita Daftar disini
    • Verifikasi email yang dikirim dari openshift selanjutnya Login Openshift
    • Create Application
    • Pilih aplikasi yang akan dibuat, contohnya Jekyll
    • Isi public URL dengan yang kita inginkan,kemudian klik create application
    • Aplikasi sudah kita buat dan kita bisa cek dengan mengakses public url yang kita isi tadi
    • Generate public key dengan menjalankan Putty
    • isi key passphrase dengan yang mudah diingat, misal "pemograman4"
    • klik save private key dan public key simpan di folder yang mudah diakses kembali
    • sebelum membuat set ssh kita Copy text yang ada di textbox public key ke ssh public key di akun openshift kita
    • memberi nama pada key nya misalnya ssh-key dan paste text yang tadi dicopy pada textbox di bawahnya
    • klik create untuk membuat ssh key dan sukses berikut tampilannya
    • Clone source dari openshift kita perlu mengetahui akses URL ssh dan mencopy alamat ULR tersebut, berikut tampilannya
    • Pada tampilan git clone masukkan URL dari source kode ssh openshift tadi selanjutnya ceklis Load putty key cari file private key yang tadi kita simpan.
    • masukkan passphrase yang tadi kita isi yaitu "Pemograman4"
    • selanjutnya di klik OK , maka akan melakukan proses cloning latihanhosting yang saya buat dari platform openshift

    • Push sukses, lakukan cara seperti di atas pada file-file lainnya yang akan di upload ke openshift
    • Setelah push sukses, kita coba akses kembali aplikasi yang kita buat tadi
    • masuk ke akun openshift kita atau klik di aplikasi
    • Membuat Artikel berhasil di buat

    Tahapan Beli Domain

    • order pembelian domain di dan cek ketersediaan domain yang kita inginkan Daftar Domain
    • Registrasi dan mendaaftarkan dan ada beberapa biodata kita untuk memvalidasi supaya bisa login
    • ketik nama domain kita misalkan amanpandawa untuk periksa kesediaan domain apakah bisa di order
    • Mengetahui Ringkasan Pemesanan yang akan kita order dan melakukan pembayaran atau Tranfer Lewat ATM
    • Menyelesaikan pemesanan dengan menceklis dan menyetujui order yang kita pesan, berikut tampilannya
    • Menerima faktur online untuk melakukan transfer pembayaran lewat ATM atau Internet Banking, berikut tampilan faktur yang telah di proses
    • setelah melakukan pembayaran lalu kita melakukan konfirmasi dengan mengirim bukti transfer atau struk bukti pembayaran meliputi Email,Nomor Faktur, Domain, nama pemilik Rekening, serta banyaknya jumlah dana yang kita transfer

    • Masuk menu Domain di alat pengelolaan selanjunya mengubah nameservers terdahulu di area clien di manage nameservers
    • Masuk menu Domain di alat pengelolaan selanjunya kelola domain
